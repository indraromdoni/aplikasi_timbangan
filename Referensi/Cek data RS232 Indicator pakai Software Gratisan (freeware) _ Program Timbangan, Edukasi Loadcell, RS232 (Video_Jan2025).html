Sebelum menggunakan atau memesan program aplikasi timbangan, saya sarankan untuk mengecek, apakah indicator timbangan anda bisa di koneksi dengan komputer dengan baik via RS232

Untuk Cek Data RS232 dari timbangan, bisa juga mendownload Program Aplikasi Jembatan Timbang Versi.17A (Demo, Portable)


setelah anda download, Extact All, Jalankan TRUKDeMo17A.EXE
nah di menu "Utility" ada Menu "Serial Port, Baris Laporan, Jenis Tiket"
click tombol "Terminal untuk Cek data RS232 dari timbangan"
nah dari situ anda bisa mencoba coba sendiri...

Selain program di atas, juga ada di internet
Pada intinya, jika management berat indicator anda bisa dilakulan dengan program timbangan di komputer, maka fungsi dari tombol tombol yang ada di indicator anda tidak berfungsi kecuali tombol ZERO
Nah, seberapa mahal indicator anda ? , seberapa fanatik anda dengan suatu merk terkenal ? semua akan sia sia, jika sudah terkoneksi dengan komputer, dan semua di management program.

Dari pengalaman saya dari tahun 1996 sampai sekarang, saya lebih menyukai indicator yang menggunakan baterai 6 Volt , kenapa ?
Coba anda lihat sertifikat spesifikasinya jika membeli suatu LoadCell, di situ ada Max tegangan yang di ijinkan, yang pernah saya tahu di range 12 V s/d 24 Vdc
Nah jika indicator anda rusak, dia akan berhenti pada tegangan baterai, jadi LoadCell anda Aman
kecuali kena petir ya...ok itu hanya pendapat saya saja....

Indicator yang menggunakan baterai ini sekarang sudah banyak di jual di suplier timbangan
Contohnya: SGW3015, A12E, A9  dll

Oh..ya langsung saja....kembali ke topik

Yang Pertama pastikan di komputer anda ada port COM nya, cek DeviceManager, itu COM1


Pastikan indicator anda terkoneksi dengan COM1, lihat manual book nya

Yang ke dua Download software gratis, di sini saya pakai Eltima demo atau PuTTy

Eltima di      www.eltima.com

install, jalankan, lalu ikuti

Nah tampil data rs232 nya, itu format type AND4329, GST-9600, SGW3015, PRESICA

==========================================================

Saya lebih suka pakai  PuTTy
di sini contoh untuk 3 indicator: SGW3015S, A12E, A9   perhatian

Download PuTTy
www.putty.org 
https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html

Pilih MSI windows Instaler, Pilih 32bit atau 64bit

Install, Jalankan

Jika anda tidak tahu baudrate indicator, jalanka berulang ulang putty dengan menginput kemungkinan baudrate:  300,600,1200,2400,4800,9600   dst, untuk mendapatkan data yang berpola dan bisa dilihat.

jika pada Window hitam tidak ada tampilan sama sekali, bisa di pastikan sinyalnya tidak ada, cek koneksi dan buku manualnya.

Ok pastikan indicator anda nyala dan rs232 ter koneksi di com1 mode Stream/Continous
Nanti akan tampil seperti ini

Koneksi PIN
SGW3015  -----------   PC Com1
Tx     2  -----------------  2 Rx
Gnd   5 -----------------  5 Gnd


A12E   -----------   PC Com1
Tx     5  -----------------  2 Rx
Gnd   1 -----------------  5 Gnd


A9   -----------   PC Com1         contoh ini, printer builtin tidak terpasang/tidak fungsi
Tx     7  -----------------  2 Rx
Gnd   8 -----------------  5 Gnd

Nah itu tidak menutup kemungkingan untuk merk lain, pada prinsipnya sama...

Ok, mudahkan, setelah pasti indicator anda bisa terkoneksi dengan PC, baru anda bisa memesan program timbangan ini...
